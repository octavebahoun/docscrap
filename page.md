# Introduction à React
React est une bibliothèque JavaScript pour créer des interfaces utilisateur. Elle permet de créer des composants réutilisables et de gérer facilement les mises à jour de l'interface utilisateur.

## Fonctions côté client
Les fonctions côté client de React permettent de créer des applications web interactives. Les principales fonctions côté client sont :

* `render()`
* `unmountComponentAtNode()`

### createRoot
La fonction `createRoot()` permet de créer un nouveau composant racine pour l'application.

### hydrateRoot
La fonction `hydrateRoot()` permet d'hydrater un composant racine existant avec des données serveur.

## Fonctions côté serveur
Les fonctions côté serveur de React permettent de créer des applications web avec un serveur. Les principales fonctions côté serveur sont :

* `renderToNodeStream()`
* `renderToPipeableStream()`
* `renderToReadableStream()`
* `renderToStaticMarkup()`
* `renderToStaticNodeStream()`
* `renderToString()`

## Les règles de React
Les règles de React sont des principes fondamentaux pour créer des applications web avec React. Les principales règles sont :

* Les composants et les Hooks doivent être des fonctions pures.
* React appelle les composants et les Hooks.
* Les règles des Hooks.

## React Server Components
Les composants serveur de React permettent de créer des applications web avec un serveur. Les principales fonctionnalités sont :

* Composants Serveur
* Actions Serveur

Note : Certaines fonctionnalités sont disponibles dans la version Canary de React.
